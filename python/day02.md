## 2. 문자열
> 문자열 생성
- 생성법 3가지
    1. 큰따옴표 양쪽으로 둘러싸기 ""
        - 문자열 내 작은따옴표(')를 쓰고 싶을때 사용
    2. 작은따옴표 양쪽으로 둘러싸기 ''
        - 문자열 내 큰따옴표(")를 쓰고 싶을때 사용
    3. *큰따옴표 3개를 연속으로 둘러싸기 """ """*
    4. *작은따옴표 3개를 연속으로 둘러싸기 ''' '''*
- 작은따옴표 and 큰따옴표 문자열에 포함시키는 방법
    - 백슬래시(\ ) 사용
    - 작은따옴표 또는 큰따옴표 앞에 사용하게 될경우 뒤에 따옴표는 문자 그대로로 인식됨.
- 여러 줄인 문자열을 변수에 대입하는 방법
    1. \n 삽입
    2. *큰따옴표 또는 작은따옴표 연속 3개 사용*
- 이스케이프 코드 -> \n, `\\`, `\'`, `\"` 이정도만 사용함.
    : 프로그래밍할 때 사용할 수 있도록 미리 정의해둔 *문자 조합.*
    - 주로 출력물을 보기 좋게 정렬하기 위한 용도로 많이 쓰인다.
- *문자열 연산*
    : 파이썬에서는 문자열을 더하거나 곱할 수 있다.
    - 더하기 연산 가능
    - 문자열 곱하기 -> "이름" * 2 = "이름이름" -> * : 문자열 반복횟수 지정
- *문자열 길이 구하기*
    - len(변수명) : 공백 포함.
- 문자열 인덱싱 -> 각 문자열 인덱스에 해당하는 문자를 출력하는 것.
    - 문자열 자릿수는 *0*부터 시작.
    - [인덱스] : 인덱스 연산자, 해당 인덱스의 문자를 출력.
    - *마이너스(-)는 끝자리부터 매긴다*. -> -1 = 문자열의 가장 끝의 문자
- *문자열 슬라이싱*
    - [begin num : end num] : end num은 포함 X
    - [begin num : ] : 끝번호 생략 시 그 문자열의 끝까지 출력.
    - [ : end num] : 첫번호 생략 시 그 문자열의 처음부터 출력.
    - [ : ] : 모든 문자열 출력.
- 문자열 자료형은 내부 문자 요소를 바꿀 수 없다.(immutable 자료형)
    -> 문자열 중간을 변경하고자 한 경우, *문자열 슬라이싱*하여 중간 문자를 바꿔 더하기 연산을 사용한다.
- 문자열 포맷팅
    - 같은 문자열에 하나의 특정한 값만 바꿔줘야 할 경우 사용.
    1. 숫자 바로 대입  
    ```
    >>> "I eat %d apples." % 3  
    'I eat 3 apples.'
    ```
    2. 문자열 바로 대입
    ```
    >>> "I eat %s apples." %"five"
    'I eat five apples.'
    ```
    3. 두개 이상의 값 넣기
    ```
    >>> number = 10
    >>> day = five
    >>> "I ate %d apples. so I was sick for %s days." %(number,day)
    'I ate 10 apples. so I was sick for five days.'
    ```

- 포맷 코드와 숫자 함께 사용하기
    1. 정렬과 공백
    ```
    >>> "%10s" % "hi"
    '        hi
    ```
    - %10s의 의미는 전체 길이가 10인 문자열 공간에 값을 오른쪽으로 정렬하고 그 앞의 나머지는 공백으로 남긴다는 의미.
    - %-10s는 위와 유사한 코드이나 값을 왼쪽으로 정렬한다는 의미.
    2. 소숫점 표현
    ```
    >>> "%10.4f" % 3.42134234
    '    3.4213'
    ```
    - 위 예는 숫자 3.42134234를 소수점 네 번째 자리까지만 표시하고 전체 길이가 10개인 문자열 공간에서 오른쪽으로 정렬하는 예를 보여 준다.

- format 함수를 사용한 포맷팅
    - 숫자 바로 대입
        ```
        >>> "I eat {0} apples".format(3)
        'I eat 3 apples'
        ```
    <br><br>
    - 문자열 바로 대입
        ```
        >>> "I eat {0} apples".format("five")
        'I eat five apples'
        ```
    <br><br>
    - 두 개 이상 넣기
        ```
        >>> number = 10
        >>> day = "three"
        >>> "I ate {0} apples. so I was sick for {1} days.".format(number, day)
        'I ate 10 apples. so I was sick for three days.'
        ```
    <br><br>
    - 이름으로 넣기
        ```
        >>> "I ate {number} apples. so I was sick for {day} days.".format(number=10, day=3)
        'I ate 10 apples. so I was sick for 3 days.'
        ```
        - {name} 형태를 사용할 경우 format 함수에는 반드시 name=value 와 같은 형태의 입력값이 있어야만 한다.<br><br>

    - 정렬
        - 왼쪽 : "{0:<10}".format("h") ->문자열 길이 10에 왼쪽으로 값 정렬
        - 오른쪽 : {0:>10}
        - 가운데 : {0:^10}<br><br>
    - 공백채우기
        - 정렬 시 공백 대신 지정 문값으로 채워넣는것.
        - {0:=^10} -> ===={0}====
        - {0:!<10} -> {0}!!!!!!!<br><br>
    8. 소수점 표현
        - {0:0.4f}
        - {0:10.4f}

- f 문자열 포맷팅
    - 파이썬 3.6버전부터 가능.
    - f'문자열 {변수}' -> 이런 방식으로 변수의 값을 참조할 수 있다.
    - f 문자열 포맷팅의 표현식
        - 변수와 +,-와 같은 수식도 함께 사용이 가능
    - 딕셔너리와 f 포맷팅
        ```
        >>> d = {'name':'홍길동', 'age':30}
        >>> f'나의 이름은 {d["name"]}입니다. 나이는 {d["age"]}입니다.'
        '나의 이름은 홍길동입니다. 나이는 30입니다.'
        ```
    - 정렬
        f'{"hi":<10}'
        f'{"hi":>10}'
        f'{"hi":^10}'
        - 공백 채우기
        f'{"hi":!^10}'
    - 소숫점 표현
        y=3.123
        f'{y:0.4f}' 
        f'{3.13:0.1f}'

- 문자열 관련 함수
    - 문자열 내장 함수.
    1. count('문자')
        - 문자열 안에 문자의 갯수 반환.
    2. find('문자')
        - 문자열 안에 문자가 *처음*나온 위치를 반환.
        - 존재하지 않는 문자는 *-1* 반환
        - 인덱스는 0부터임.
    3. index('문자')
        - 문자열 안에 문자가 처음나온 위치를 반환.
        - find() 차이점 -> 존재하지 않는 문자의 경우, 오류가 발생함.
    4. join('문자열')
        ```
        >>> ",".join('abcd')
        'a,b,c,d'
        ```
        - abcd 문자열 사이에 ,를 넣어줌.
        - 튜풀이나 리스트도 입력으로 올 수 있음.
    5. upper()
        - void, 소문자 -> 대문자, 다 대문자면 변화 X
    6. lower()
        -void, 대문자 -> 소문자, 다 대문자면 변화 X
    7. lstrip()
        - 문자열 중 가장 왼쪽에 있는 한칸 이상의 연속된 공백을 지운다.
    8. rstrip()
        - 문자열 중 가장 오른쪽에 있는 한칸 이상의 연속된 공백을 지운다.
    9. strip()
        - 문자열의 양쪽에 있는 한칸 이상의 연속된 공백을 지운다.
    10. replace('문자열1','문자열2')
        - 문자열1 기존문자열에서 문자열2 다른 문자열로 변경.
    11. split()
        - 문자열 나누기
        - 괄호 안이 비어있는 경우, 문자열을 공백 기준으로 나누어 리스트에 담는다.
        - 문자가 있는 경우, 문자를 구분자로 하여 문자열을 나누어 리스트에 담는다.

## 3. 리스트 자료형
- 숫자나 문자열 모음을 쉽게 표현할 수 있는 자료형.
- `리스트명 = [요소1, 요소2, 요소3, ...]` 
- 리스트 안에는 어떠한 자료형이든 섞어서 포함시킬수 있다. 문자열 + 숫자형

> 리스트의 인덱싱과 슬라이싱
1. 리스트 인덱싱
- 문자열과 마찬가지로 리스트의 첫요소의 인덱스는 0이다.
- 값을 가져오는 것은 인덱스 연산자([])로 가져온다.
    ```  
    EX) >>> a = [1, 2, 3]
        >>> a[0]
        1
    ```
- a[-1]은 문자열에서와 마찬가지로 리스트 a의 마지막 요솟값을 말한다.
- 리스트 안에 리스트 자료형을 또 추가할 수 있음.
    ```
    >>> a = [1, 2, 3, ['a', 'b', 'c']] 
    >>> a[-1]
    ['a', 'b', 'c']
    >>> a[3]
    ['a', 'b', 'c']
    ```
- 리스트 안에 리스트 요소 끄집어 내기
    ```
    >>> a[-1][0]
    'a'
    ```
- 리스트를 다중으로 중첩해서 사용하는 것은 혼란스럽기 때문에 자주 사용하지는 않는다.

2. 리스트 슬라이싱
-  문자열에서 했던 것과 사용법이 완전히 동일하다.
    ```
    >>> a = [1, 2, 3, 4, 5]
    >>> a[0:2]
    [1, 2]
    ```
- 중첩된 리스트에서 슬라이싱
    ```
    >>> a = [1, 2, 3, ['a', 'b', 'c'], 4, 5]
    >>> a[2:5]
    [3, ['a', 'b', 'c'], 4] # 리스트 내부 리스트 요소도 하나당 하나의 인덱스로 침. -> 총 요소 5개 
    >>> a[3][:2]
    ['a', 'b']
    ```

> 리스트 연산하기
1. 리스트 더하기(+)
    ```
    >>> a = [1, 2, 3]
    >>> b = [4, 5, 6]
    >>> a + b
    [1, 2, 3, 4, 5, 6]
    ```
2. 리스트 반복하기(*)
    ```
    >>> a = [1, 2, 3]
    >>> a * 3
    [1, 2, 3, 1, 2, 3, 1, 2, 3]
    ```
3. 리스트 길이구하기(len)
- 문자열, 리스트, 튜플, 딕셔너리 자료형에 길이를 구하는 함수 len()
    ```
    >>> a = [1, 2, 3]
    >>> len(a)
    3
    ```  
※ 파이썬에서는 자료형이 다른 값끼리 연산이 불가능하다.  
```
문자열과 정수형간의 연산
>>> a = [1, 2, 3]
>>> a[2] + "hi"

# 오류 발생
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
TypeError: unsupported operand type(s) for +: 'int' and 'str'

# 연산을 위해 정수형을 문자열로 바꿔줌.(내장함수 str())
>>> str(a[2]) + "hi"
3hi
```  

> 리스트의 수정과 삭제
1. 리스트 값 수정
    ```
    >>> a = [1, 2, 3]
    >>> a[2] = 4
    >>> a
    [1, 2, 4]
    ```
2. del함수 사용하여 리스트 요소 삭제
    ```
    >>> a = [1, 2, 3]
    >>> del a[1]
    >>> a
    [1, 3]
    ```
    - 'del 객체' 사용하는 내장 함수
    - 파이썬에서 객체는 사용되는 모든 자료형을 의미한다.
- 슬라이싱처럼 여러요소 한번에 삭제
    ```
    >>> a = [1, 2, 3, 4, 5]
    >>> del a[2:]
    >>> a
    [1, 2]
    ```

> 리스트 관련 함수들
- 문자열과 마찬가지로 변수명 뒤에 '.'를 붙여서 함수를 사용한다.

1. append() : 리스트에 요소 추가
    ```
    >>> a = [1, 2, 3]
    >>> a.append(4)
    >>> a
    [1, 2, 3, 4]
    ```
2. sort() : 리스트 정렬
- 기본 : 오름차순(숫자, 문자)
    ```
    >>> a = [1, 4, 3, 2]
    >>> a.sort()
    >>> a
    [1, 2, 3, 4]
    ```
3. reverse() : 리스트 뒤집기
-  리스트 요소들을 순서대로 정렬한 다음 다시 역순으로 정렬하는 것이 아니라 그저 현재의 리스트를 그대로 거꾸로 뒤집는다.
    ```
    >>> a = ['a', 'c', 'b']
    >>> a.reverse()
    >>> a
    ['b', 'c', 'a']
    ```
4. index() : 인덱스 반환
- 값의 인덱스를 반환.
    ```
    >>> a = [1,2,3]
    >>> a.index(3)
    2
    >>> a.index(1)
    0
    ```
- 리스트에 없는 값을 넣을 경우 오류 발생.

5. insert() : 리스트 요소에 삽입
- insert(a,b) : 리스트 인덱스 a 위치에 b라는 값을 넣는다.
    ```
    >>> a = [1, 2, 3]
    >>> a.insert(0, 4)
    >>> a
    [4, 1, 2, 3]
    ```
6. remove() : 리스트 요소 제거
- remove(x)는 리스트에서 첫 번째로 나오는 x를 삭제하는 함수이다.
    ```
    >>> a = [1, 2, 3, 1, 2, 3]
    >>> a.remove(3)
    >>> a
    [1, 2, 1, 2, 3]
    ```
7. pop() : 리스트 요소 끄집어내기
- pop()은 리스트의 맨 마지막 요소를 리턴하고 그 요소는 삭제한다.
    ```
    >>> a = [1,2,3]
    >>> a.pop()
    3
    >>> a
    [1, 2]
    ```
- pop(x)는 리스트의 인덱스 x번째 요소를 리턴하고 그 요소는 삭제한다.
    ```
    >>> a = [1,2,3]
    >>> a.pop(1)
    2
    >>> a
    [1, 3]
    ```
8. count() : 리스트에 포함된 요소 x의 갯수 세기
- count(x)는 리스트 안에 x가 몇 개 있는지 조사하여 그 개수를 리턴하는 함수이다.
    ```
    >>> a = [1,2,3,1]
    >>> a.count(1)
    2
9. extend() : 리스트의 확장
- extend(x)에서 x에는 *리스트*만 올 수 있으며 원래의 a 리스트에 x 리스트를 더하게 된다.
    ```
    >>> a = [1,2,3]
    >>> a.extend([4,5])
    >>> a
    [1, 2, 3, 4, 5]
    >>> b = [6, 7]
    >>> a.extend(b)
    >>> a
    [1, 2, 3, 4, 5, 6, 7]
    ```
- a.extend([4, 5])는 a += [4, 5]와 동일하다.  
` a += [4, 5]는 a = a + [4, 5]와 동일한 표현식이다.`  

## 4. 튜플 자료형
- 튜플(tuple)은 리스트와 거의 동일한 자료형이며 몇가지 차이점이 있다.
    - 리스트 [] / 튜플 ()
    - *리스트는 요소 생성, 수정, 삭제가 가능하지만 튜플은 요소 값을 바꿀 수 없다.*
        - 지우거나 수정 시 오류발생.
    - 차이점 예시
    ```
    >>> t1 = ()
    >>> t2 = (1,)
    >>> t3 = (1, 2, 3)
    >>> t4 = 1, 2, 3
    >>> t5 = ('a', 'b', ('ab', 'cd'))
    ```
    - t2 처럼 하나의 요소만 들어가도 뒤에 (,)쉼표를 반드시 붙여야 한다.
    - t4 처럼 괄호 생략이 가능하다.

> 튜플 다루기
1. 인덱싱
    - 문자열 인덱싱과 동일함.
2. 슬라이싱
    - 튜플 요소 잘라내기
3. 더하기
    - 튜플 요소 더하기
4. 곱하기
    - 튜플 요소 반복
5. 길이 구하기
    - len()함수

## 5. 딕셔너리 자료형
- 대응 관계를 나타내는 자료형으로, 연관배열(Associative array) 또는 해시(Hash)라고 한다.
- 이를 파이썬에서 딕셔너리 라고 한다.
- 즉, Key-Value를 한쌍으로 갖는 자료형.
- 순차적(리스트, 튜플)으로 해당 요소값을 구하지 않고, Key를 통해 Value 값을 얻는다.
    - 딕셔너리에는 인덱싱이 없다.

> 딕셔너리 생성
- 기본형태
    ```
    {Key1:Value1, Key2:Value2, Key3:Value3, ...}
    ```
- key : value 형태이며, (,)쉼표로 구분된다.
- value에는 리스트 자료형도 들어갈 수 있다.

> 딕셔너리 쌍 추가,삭제
- 추가
    ```
    >>> a = {1: 'a'}
    >>> a[2] = 'b'
    >>> a
    {1: 'a', 2: 'b'}

    >>> a['name'] = 'pey'
    >>> a
    {1: 'a', 2: 'b', 'name': 'pey'}

    >>> a[3] = [1,2,3]
    >>> a
    {1: 'a', 2: 'b', 'name': 'pey', 3: [1, 2, 3]}
    ```
- 삭제 : del 함수
    ```
    >>> del a[1]
    >>> a
    {2: 'b', 'name': 'pey', 3: [1, 2, 3]}
    ```

> 딕셔너리 사용법
1. 딕셔너리에서 Key를 사용하여 Value 얻기
- 딕셔너리 변수명[key]
    ```
    >>> dic = {'name':'pey', 'phone':'010-9999-1234', 'birth': '1118'}
    >>> dic['name']
    'pey'
    >>> dic['phone']
    '010-9999-1234'
    >>> dic['birth']
    '1118'
    ```
2. 딕셔너리 만들때 주의사항
- Key 값은 중복되서는 안된다.
- Key 값으로 리스트 자료형은 사용 불가능하지만, 튜플 자료형은 사용 가능하다.
    - key 값 사용 가능여부는 변하는 값(mutable)인지 아닌지(immutable)인지에 달려있다.
    - value는 상관없다.

> 딕셔너리 관련 함수
1. Key 리스트 만들기(keys())
    ```
    >>> a = {'name': 'pey', 'phone': '010-9999-1234', 'birth': '1118'}
    >>> a.keys()
    dict_keys(['name', 'phone', 'birth'])

    >>> for k in a.keys():
    ...    print(k) # 들여쓰기 필수
    ...
    name
    phone
    birth
    ```
    - dict_keys 객체 반환. -> 리스트로 변환하지 않아도 기본적인 반복 구문에서 사용 가능함.
        - list를 사용하지 않는 이유 : 메모리 낭비를 줄이기 위함.
        - list와 거의 동일하지만 list 고유의 함수는 사용이 불가능하다.
    - 리스트로 사용해야할 경우, *list(a.keys())* 로 변환.
2. Value 리스트 만들기(values())
    ```
    >>> a.values()
    dict_values(['pey', '010-9999-1234', '1118'])
    ```
    - values 함수를 호출하면 dict_values 객체를 돌려준다.
3. Key, Value 쌍 얻기(items())
    ```
    >>> a.items()
    dict_items([('name', 'pey'), ('phone', '010-9999-1234'), ('birth', '1118')])
    ```
    - items 함수는 Key와 Value의 쌍을 튜플로 묶은 값을 dict_items 객체로 돌려준다.
4. Key: Value 쌍 모두 지우기(clear())
    ```
    >>> a.clear()
    >>> a
    {}
    ```
    - 딕셔너리 내부 모든 요소 삭제.
    - 빈 리스트를 [], 빈 튜플을 ()로 표현하는 것과 마찬가지로 빈 딕셔너리도 {}로 표현한다.
5. Key로 Value얻기(get)
    - get(x) 함수는 x라는 Key에 대응되는 Value를 리턴한다. 
    - 앞에서 살펴보았듯이 a.get('name')은 a['name']을 사용했을 때와 동일한 결괏값을 리턴한다.
    - 차이점은 a['key'] 방식으로 존재하지 않는 키값을 조회 시 오류가 발생하고, get을 사용하면 None(거짓)을 반환한다.
    - 딕셔너리 안에 찾으려는 Key가 없을 경우 미리 정해 둔 디폴트 값을 대신 가져오게 하고 싶을 때에는 get(x, '디폴트 값')을 사용하면 편리하다.
    ```
    >>> a = {'name':'pey', 'phone':'010-9999-1234', 'birth': '1118'}
    >>> print(a.get('nokey'))
    None
    >>> print(a['nokey'])
    Traceback (most recent call last):
    File "<stdin>", line 1, in <module>
    KeyError: 'nokey'

    >>> a.get('nokey', 'foo')
    'foo'
    ```
7. 해당 Key가 딕셔너리 안에 있는지 조사하기(in)
    - 딕셔너리 안에 해당 Key가 있는 경우 true, 없는 경우 false를 반환한다.
    ```
    >>> a = {'name':'pey', 'phone':'010-9999-1234', 'birth': '1118'}
    >>> 'name' in a
    True
    >>> 'email' in a
    False
    ```

## 6. 집합 자료형
- 파이썬 2.3 버전 이후부터 지원한 자료형
- set 키워드를 사용한다.
    ```
    # 리스트 넣기
    >>> s1 = set([1,2,3])
    >>> s1
    {1, 2, 3}

    # 문자열 넣기
    >>> s2 = set("Hello")
    >>> s2
    {'e', 'H', 'l', 'o'} # 중복불가 및 순서 없음
    ```

> 집합 자료형의 특징
- 중복 허용하지 않음
- 순서가 없음(인덱싱이 없음)
    - 인덱싱으로 접근하기 위해서는 list 또는 tuple로 변환해야한다.

> 교집합, 합집합, 차집합 구하기
- 교집합 : & 기호, set1 & set2
- 합집합 : | 기호 또는 union() 함수, set1 | set2 또는 set1.union(set2)
- 차집합 : - 기호 또는 difference() 함수, set1 - set2 또는 set1.difference(set2)

> 집합 자료형 관련 함수
### 값 1개 추가 (add)
- 1개의 값만 추가시 사용하는 함수
    ```
    >>> s1 = set([1, 2, 3])
    >>> s1.add(4)
    >>> s1
    {1, 2, 3, 4}
    ```
### 값 여러 개 추가 (update)
- 여러개의 값을 한번에 추가시 사용하는 함수
- update 괄호 안에 리스트자료형이 들어갈 수 있음.
    ```
    >>> s1 = set([1, 2, 3])
    >>> s1.update([4, 5, 6])
    >>> s1
    {1, 2, 3, 4, 5, 6}
    ```
### 특정 값 제거하기(remove)
- 특정 값 제거하기
    ```
    >>> s1 = set([1, 2, 3])
    >>> s1.remove(2)
    >>> s1
    {1, 3}
    ```

## 7. 불 자료형
- 불(bool) 자료형은 참(true)과 거짓(false) 두가지 값만 가진다.
- 첫자를 항상 대문자로 사용할 것. True False

> 자료형의 참과 거짓

| 값         | 참 or 거짓 |
|-----------|---------|
| "python"  | 참       |
| ""        | 거짓      |
| [1, 2, 3] | 참       |
| []        | 거짓      |
| (1, 2, 3) | 참       |
| ()        | 거짓      |
| {'a': 1}  | 참       |
| {}        | 거짓      |
| 1         | 참       |
| 0         | 거짓      |
| None      | 거짓      |

- 문자열, 리스트, 투플, 딕셔너리 가 비어있으면 false가 나옴.
- 숫자형은 값이 0일 때 false가 나옴.
- None은 false를 뜻한다.

> 자주 사용되는 구문
- while 자료형 : ...
    ```
    >>> a = [1, 2, 3, 4]
    >>> while a:
    ...     print(a.pop())
    ...
    4
    3
    2
    1
    ```
- 자료형이 list인 경우, 반복문 내부에 요소를 지우는 pop() 함수를 사용했을 경우,
- 마지막 요소가 빠져나가게 되면 리스트가 비게 되어 False를 반환하며, 이로인해 반복문을 탈출한다.

> bool 연산
- 문자열
    ```
    >>> bool('python')
    True

    >>> bool('')
    False
    ```

## 8. 변수
- 파이썬에서 변수는 객체이다.
- 만약 a = [1, 2, 3]이라고 하면, [1, 2, 3] 값을 가지는 리스트 자료형 데이터(객체)가 자동으로 메모리에 생성되고 변수 a는 [1, 2, 3] 리스트가 저장된 *메모리의 주소*를 가리키게 된다.
