2023.02.07 SQL 복습 및 연습문제
- 외부조인
 : 동등조인으로 구하지 못하며, 한쪽에는 있고 다른쪽에는 없는 데이터를 검색하고
   싶을 때 사용함. 조건에 맞지 않아도 해당 칸의 값을 출력하고 싶은 경우 사용.

- 조인 문법
1. 일반 조인(동등 조인)
SELECT 속성명
FROM 테이블명
WHERE 조인 조건 AND 검색 조건

SELECT 속성명
FROM 테이블명
INNER JOIN 테이블명 ON 조인조건
WHERE 검색 조건

2. 외부 조인
SELECT 속성명
FROM 테이블명
LEFT | RIGHT (OUTER) JOIN 테이블명
ON 조인 조건
WHERE 검색 조건

- 부속질의 : 질의문 안에 질의문
1. 스칼라 부속질의 : 단일값만 검색 / SEELCT절
2. 인라인 뷰 부속질의 : 가상의 테이블, 여러 레코드 검색 / FROM절
3. 중첩 질의 : 비교 연산 -> 스칼라값 / 집합 연산 시 -> 여러 레코드 / WHERE절
