INSERT문
	- 테이블에 새로운 투플을 삽입하는 명령어이다.
	- 행추가
INSERT INTO 테이블명[(속성명1,2,3, ...)]
	VALUES(값 리스트);
	- 새로운 투플 삽입 시 속성명은 생략이 가능한 대신, 데이터 입력 순서는 속성의 순서와 일치해야 한다.
	- 데이터는 항상 속성의 순서대로 입력하지 않아도 된다. 속성 순서를 바꾸면 입력 순서도 맞춰서 바꿔줘야 한다.
	- 만약, 몇 개의 속성만 입력 시 해당 속성 입력값만 명시해주면 된다.
	- 비워진 값은 null이 채워진다.
	★ INSERT문은 SELECT문을 사용하여 작성 가능하며, 이는 다른 테이블의 속성이 해당 테이블과 같을 경우
	    안에 있는 투플을 모두 삽입하는 방식이다.(대량 삽입,BULK INSERT)
INSERT INTO 테이블명(속성명1,2,3,...)
	SELECT 속성명1,2,3... FROM 테이블명
