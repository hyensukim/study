DML - 2일차 ORDER BY & 집계함수
- ORDER BY 속성명 ASC(오름차순) DESC(내림차순)
- 집계함수
	집계 : 통계와 비슷한 의미, 데이터 베이스에서 사용하는 통계.
	집계를 위해 GROUP BY문을 사용하고, 구체적인 집계 내용은 집계함수를 사용한다.
- AS 키워드
	집계 후 결과 열의 제목을 붙혀주는 것.
	 최근 버전의 DBMS에서는 AS 생략이 가능하다.
SUM : 합계
AVG : 평균
MAX : 최대값(투플)
MIN : 최소값(투플)
COUNT : 행(투플)의 갯수
	- NULL 값은 제외.
