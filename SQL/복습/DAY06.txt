2022.02.06 SQL 복습 및 연습문제

두 개 이상 테이블에서 SQL 질의
- SQL은 여러 개의 테이블을 질의하는 두가지 방법을 제공한다.
- 조인과 부속질의

조인
- 조인은 한 테이블의 행을 다른 테이블의 행에 연결하여 두개 이상의 테이블을 결합하는 연산.
- 카티전 프로덕트 연산(단순 순서쌍)
    : 두 테이블을 아무런 조건없이 SELECT할 경우 되는 형태.
    -> 논리에 맞지 않는 결과가 출력됨.
- 위의 카티전 프로것트 연산은 WHERE 절에 두 테이블의 연결 조건을 추가함으로서 해결 가능하다.
- 두 개 이상 테이블에서 질의 시, 각 열 이름 앞에 테이블명을 명시하여 구분해줘야 한다.
EX) madang.Customer.custid = madang.Orders.custid -- madang은 스키마 명시코자 사용한것이므로 신경 X
- 여러 개의 테이블을 연결하여 하나의 테이블을 만드는 과정을 테이블 조인이라고 한다.
- 위의 예시처럼 동등한 열이 있는 경우 그 열을 통해 연결한 것을 동등 조인이라 한다.
- 조인을 통해 질의한 결과는 단일 테이블이며, 이 테이블은 SQL 문법(WHERE, GROUP BY, ORDER BY ...) 사용이 가능하다.
- 세 개 이상의 테이블 조인도 가능하다.

정리
두 개 이상 테이블에서 SQL 질의
- 조인
- 부속 질의

조인
 - 테이블 2개 이상을 단일 테이블로 만드는 과정.
 - 동등 조인을 통해 가능.(동등한 열을 통해 두 테이블을 하나로 연결함.)
 - 조인을 통해 만들어진 단일 테이블에도 SQL 문법 사용이 가능.
 - 세 개 이상의 테이블도 하나로 조인 가능.(하나의 매개체가 되는 테이블이 있을 경우.)
